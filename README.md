# 🌧 Rain Prediction and Forecasting using Machine Learning

This project is a Machine Learning-based web application that predicts whether it will rain or not based on historical weather data.

The system uses meteorological parameters such as:

- Temperature  
- Humidity  
- Atmospheric Pressure  
- Wind Speed  

A Logistic Regression model is trained using scikit-learn and integrated into a Flask web application to provide real-time predictions through a user-friendly interface.

## 🚀 Features
- Binary Rain / No Rain prediction
- Machine Learning model trained on weather dataset
- Flask backend integration
- Clean HTML & CSS frontend
- Saved model using pickle
- End-to-end ML workflow (Training → Evaluation → Deployment)

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- HTML & CSS
